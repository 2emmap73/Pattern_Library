Typography

Typography is the visual component of any website. It is needed for the users of the site to read the information displayed on the website.

Usage

Typography is used as a medium of communication between the website and its users. When a user visits a website, typography is used to determine what information the website provides and wheter it is worth staying on the site. Some websites use a system of font stack instead of a custom font, this eliminates the use of external resources. The applications may look slightly different as they adapt to the different screen sizes but that's ok.

font-family: -apple-system, 
             BlinkMacSystemFont, 
             "Segoe UI", 
             Roboto, 
             Oxygen-Sans, 
             "Helvetica Neue"

Typographic Scale

Typography scale determines how the user would understand the website and the meaning of the content displayed. For example: a Title of the page will have a different font size compared to the text underneath it.

Usage

As mentioned above in the example, the user must be able to distingish a title from the content, this is done by the use of font-size. Hence, the typographic scale is used to create a balanced and harmonious font sizing. The changes in the font size depends on various factors, one of them being adpating to diffent screen sizes.
