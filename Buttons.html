Buttons

Default Button

When users first look at a particular page which contains more than one or just one button, they are looking at a default button. These default buttons wait for the user to interact with them.

Usage

Default Buttons are to call an action such as, when a user clicks on the 'Search' button on any website that has a search tab, the user gets all the results of their search terms.

HTML 

 
	<div id = "preview">
	  <button class = "button"> Default Button </button>
	</div>

CSS 

 
	.button {
		padding: 20px 34px;
		background-color: #b90060;
		color: white;
		font-weight: 700;
		border: none;
		display: block;
		margin-left: auto;
		margin-right: auto;
	}
	.button:hover {
		background-color: white;
		border: 2px solid #b90060;
		color: purple;
	}

Ghost Button

Ghost Buttons are empty, transparent buttons which have basic shape like square and rectangle. Sometimes, Ghost buttons are also used instead of Default buttons.

Usage

The most effective way of using a Ghost Button is to ensure that the user follows a principle of looking at the website from left to right. This would ensure that the last thing the user would do is to click on the Ghost button to call an action.

HTML 

 
	<div id = "Ghost-button">
	  <button class = "ghost-button"> Ghost Button </button>
	</div>

CSS 

 
	.ghost-button {
		padding: 20px 34px;
		background-color: white;
		color: purple;
		font-weight: 700;
		border: 2px solid purple;
		display: block;
		margin-left: auto;
		margin-right: auto;
	}
	.ghost-button:hover {
		background-color: #4A0148;
		border: 2px solid purple;
		color: white;
}

Dropdown Button

Dropdown buttons are elements, which when clicked, displays additional contents.

Usage

The most common way to use a Dropdown button is to organise pages in one section. This is essentialy a sub-menu. Dropdown menus can also be used in blogs. Using it would increase the chance of enhancing the website's appearance by making it clean.

Link 1 Link 2 Link 3

HTML 

 
	<div class = "dropdown_button">
	   <button class = "dropdwnbtn"> Dropdown </button>
	<div class = "dropdown_content">
	   <a href = "#">Link 1 </a>
	   <a href = "#">Link 2 </a>
	   <a href = "#">Link 3 </a>
	</div> 
	</div>

CSS 

 
	.dropdwnbtn {
		background-color: #4CAF50;
		color: white;
		padding: 16px 34px;
		font-size: 16px;
		border: none;
	}

	.dropdown_button {
		position: relative;
		display: inline-block;
		margin: 0% 43%;
	}

	.dropdwndown_content {
		display: none;
		position: absolute;
		background-color: #f1f1f1;
		min-width: 160px;
		box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
		z-index: 1;
	}

	.dropdwndown_content a {
		color: black;
		padding: 12px 16px;
		text-decoration: none;
		display: block;
	}

	.dropdwndown_content a:hover {background-color: #ddd}

	.dropdown_button:hover .dropdwndown_content {
		display: block;
	}

	.dropdwnbtn:hover .dropdown_button {
		background-color: #3e8e41;
	}

Flat Button

Flat buttons are not like other buttons, they are text-only buttons.

Usage

Flat buttons are used in toolbars, inline and dialogs. These buttons do not lift, but fill with colour when pressed.

HTML 

 
	<div id = "Flat-button">
	  <button class = "flat-button"> Flat Button </button>
	</div>

CSS 

 
	#Flat-button {
		padding: 20px;
		border-top: 2px solid black;
	}
	.flat-button {
		border: none;
		background-color: white;
		color: #2c2cf9;
		font-size: 118%;
		display: block;
		margin-left: auto;
		margin-right: auto;
	}
	.flat-button:hover {
		padding: 16px 34px;
		background-color: #d8d7d7;
		color: #2c2cf9;
		border-color: #4fd620;
	}

Small Button

Buttons can come in any size. User interprets the meaning of each button size differently.

Usage

Small buttons can be used in forms such as 'cancel', 'Next'. It can also be used in smaller screens.

HTML 

 
	<div id = "Small-button">
	  <button class = "small_button"> Small Button </button>
	</div>

CSS 

 
	#small-button {
		padding: 20px;
	}
	.small_button {
		background-color: #ff4242;
		padding: 14px;
		display: block;
		margin-left: auto;
		margin-right: auto;
		border: none;
		color: white;
		font-size: 0.9em;
	}

Icon Button

All buttons perform a call of action. Buttons with icons also perform a call of action just like other buttons.

Usage

Icons used in buttons help users to know what action they are about to take place before they click that button. An example of this would be when a user is about to download a software or an image, the download button would have a 'down' arrow, giving a clear message to the user that they are about to download the file when they click the button.

HTML 

 
	<div id = "icon-button">
	  <button class = "icon_button"> Download <img src="Images/down_arrow.png" </button>
	</div>

CSS 

 
	.icon_button {
		padding: 20px;
		border: none;
		background-color: #4fd620;
		color: white;
		font-size: 118%;
		display: block;
		margin-left: auto;
		margin-right: auto;
		border-radius: 10px;
	}
	.icon_button:hover {
		background-color: #88ec66;
		color: white;
		border-color: #4fd620;
	}
