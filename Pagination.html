Pagination

What is Pagination?

Pagination divides pages into numbers to organise contents. The user has to click on the page numbers to progress further. A good example of pagination being used is on every search engine.

Usage

Page numbers are displayed at the bottom of page so that the user can read through the contents on the page easily. It makes the contents on the page the top priority for the user so tha the user reads that first.

HTML 

 
	<ul class="page_numbers">
	 <li class="pn"><a href="#">Previous</a></li>
	 <li class="pn"><a href="#" class = "number">1</a></li>
	 <li class="pn"><a href="#" class = "number">2</a></li>
	 <li class="pn"><a href="#" class = "number">3</a></li>
	 <li class="pn"><a href="#" class = "number">4</a></li>
	 <li class="pn"><a href="#" class = "number">5</a></li>
	 <li class="pn"><a href="#">Next</a></li>
	</ul>

		margin: 0% 24%;

	}
