Footer

What is a Footer?

A footer on a webpage is located at the bottom of the page and it contains some useful information about the website, social media links and it also acts as a site map of the website.

Usage

Footer acts as a secondary navigation system for the users at the bottom of the page.

  • Home
  • Blog
  • Contact Me
  • About Me

HTML 

 
	<div class="foot">
	  <div class="footer">
	   <ul class="footer_menu">
	     <li>Home</li>
	     <li>Blog</li>
	     <li>Contact Me</li>
	     <li>About Me</li>
	   </ul>
	<div class="social_Media">
	  <img src="Images/facebook.png" alt="Facebook"/>
	  <img src="Images/snapchat.png" alt="Snapcat"/>
	</div>
	</div>
	</div>

CSS 

 
	.foot {
		padding-top: 20px;
		padding-bottom: 20px;
	}
	.footer{
		background-image: linear-gradient(133deg, #1500b7 0%, #920202 100%);
		display: flex;
		justify-content: space-between;
		align-items: center;
	}
	.footer_menu li {
		text-transform: uppercase;
		color: white;
		font-weight: 700;
		margin-bottom: 16px;
		list-style: none;
	}
	.footer img {
		margin-right: 16px;
	}
