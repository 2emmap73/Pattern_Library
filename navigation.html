Navigation

Navigation bar

Navigation bar is a graphical user interface which helps the users to access information on the website.

Usage

Navigation bar can be used on the top of the page and sides (right or left) of the page. The best place to implement a navigation bar is either on the top of the page or left hand side of the page, also known as a sidebar. The preview below is showing a navigation bar which would be placed on the top of the page.

  • Home
  • Blog
  • Contact Me
  • About Me

HTML 

 
	<div class = "navO">
	  <ul class = "navpreview">
	    <li class = "naP"><a class = "active" href="#home> Home </a></li> 
	    <li class = "naP"><a class = "active" href="#blog> Blog </a></li> 
	    <li class = "naP"><a class = "active" href="#contactMe> Contact Me </a></li> 
	    <li class = "naP"><a class = "active" href="#aboutMe> About Me </a></li> 
	  </ul>
	</div>

CSS 

 
	.navpreview {
		list-style-type: none;
		padding: 0;
		overflow: hidden;
		background-color: #ca5454;
		margin-right: auto;
		margin-left: auto;
		display: block;
	}

	li.naP {
		float: left;
	}

	li.naP a {
		display: block;
		color: white;
		text-align: center;
		padding: 14px 16px;
		text-decoration: none;
	}

	li.naP a:hover {
		background-color: #da1c1c;
	}
