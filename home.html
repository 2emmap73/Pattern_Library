Introduction

This pattern library is created to showcae and display the components that are used in a portfolio website. The components showcased in this pattern library are tested and validated to make the code as high quality as possible.

With each component displayed in the pattern library, an appropiate documentation has also been provided with each component such as, their desciption and usage. This will help web developers to understand each component and where they can be used in the website.

The Aim

This pattern library aims to help web developers to easily build upon the code and design new pages. This has been ensured by following Atomic design principles during the design and building process of the pattern library.

Through the use of documentation for each component, it will make it clear and concise to the user how and where to use each component. Added to that, the code will also be showcased so that the user will know how the component was made.
