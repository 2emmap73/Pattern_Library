Jumbotron

What is a Jumbotron?

A jumbotron is a big box to get attention of the user. It usually has some important content that the user would be intersted to look at.

Usage

Jumbotron is used like a banner at the top of the website to gain attention of the user. It usually appears with a heading, paragraph and a button.

Hi, I am PE

I'm a web designer and a networking guy. Please feel free to check out about web design and network by clicking the button below.

Web Design and Networking

	<div class = "preview_jumbotron">
